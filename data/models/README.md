# Local Models

Put local model files here when running Mklan Studio on your machine.

Expected runtime subfolders include:

```text
models/
  captioning/
  images/
    Base/
    VAE/
  loras/
  video/
```

Model weights and caches are intentionally ignored by Git. This repository does
not distribute checkpoints, LoRAs, VAEs, captioning model weights, or Hugging
Face cache files.
